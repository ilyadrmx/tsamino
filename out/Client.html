onEvent(eventName, callback)

Get websocket event callbacks from event names. Available event names are: - textMessage: Messages with **text content** (message type 0) - chatMessage: Every message received from the **chat** (any of the message types) - websocketEvent: **Every** websocket event (chat message, notification, invitation, etc.)
